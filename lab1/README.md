## Вариант 12: Бюджетирование (https://about.coinkeeper.me/)

### Данные предметной области
- **Пользователь**
- **Планируемый доход**
- **Планируемый расход**

### Требования к API
- Создание нового пользователя
- Поиск пользователя по логину
- Поиск пользователя по маске имя и фамилии
- Создать планируемый доход
- Получить перечень планируемых доходов
- Создать планируемый расход
- Получить перечень планируемых расходов
- Посчитать динамику бюджета за период

---

## Роли пользователей
- **Пользователь**: регистрируется, ведёт бюджет (добавляет планируемые доходы/расходы), ищет пользователей, смотрит динамику бюджета за период.

## Внешние системы
- **Email-сервис** — отправка писем подтверждения регистрации. При создании нового пользователя (POST /users) User Service отправляет email для верификации аккаунта через внешний SMTP-сервис.
